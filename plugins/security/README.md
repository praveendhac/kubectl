# Security plugin

* Security plugin help
```
$ kubectl plugin security -h
Kubernetes Security checks based on CIS, NIST etc. Basic Security Scan

Options:
      --cis='': CIS benchmark checks
      --nist='': NIST benchmark checks
      --scan='': Basic security scan

Usage:
  kubectl plugin security [flags] [options]

Use "kubectl options" for a list of global command-line options (applies to all
commands).
```
